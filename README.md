# Lakehouse Mentorship website

https://lakehousementorship.com

Author: Lakehouse Mentorship LLC by Sergei Parshev

## How to run

Use a regular hugo server command to run it locally, for example:
```
$ hugo server --gc --bind 0.0.0.0 --baseURL http://localhost:1313
```
